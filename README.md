# amazon-history.js

1. chomeを開きます（chromeでしか試してません）
1. amazonの注文履歴を開きます https://www.amazon.co.jp/gp/css/order-history/
1. F12を押します
1. Consoleタブを開きます
1. ソースをすべてコピーしてConsoleに貼り付けます
